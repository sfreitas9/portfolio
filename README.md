# Sherry Freitas' Portfolio

This project contains a simple portfolio of the work I've done at San Diego Code School in the summer of 2019.

To install:
```
npm install
```

To run:
```
node server.js
```
Then go to http://localhost:8080 in your browswer.

To test:
```
npm test
```
